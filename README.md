
# Fares Mohamed | Portfolio

Welcome to my personal portfolio! This project showcases my work, experience, and skills as a Backend .NET Developer, built with React, Styled-Components, and Framer Motion for smooth, modern animations.

## 🚀 Live Demo

Check out the live portfolio: [fares-ma.github.io/portfolio](https://fares-ma.github.io/portfolio)

## 🛠️ Features

- Modern, responsive design
- Smooth page transitions and animations
- Projects section with real-world work
- About, Experience, and Skills pages
- Dark/Light theme toggle
- Custom SVG graphics and icons
- Fast performance with React 18

## 📦 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/fares-ma/portfolio.git
cd portfolio
npm install
```

Run locally:

```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 Customization

- Update your info in `src/data/` and `src/components/` as needed.
- Replace images in `src/assets/Images/`.
- Edit theme/colors in `src/components/Themes.js` and `src/globalStyles.js`.
